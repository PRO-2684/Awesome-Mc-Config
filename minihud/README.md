# 🧭 Minihud
> It is recommended to use this mod together with Xaero's minimap.

## 🔗 Mod link
[CurseForge](https://www.curseforge.com/minecraft/mc-mods/minihud)

## 📂 Config file path
`./config`

## 💡 Features
- FPS display
- Real date & time
- Coordinate
- Block coordinate
- Facing
- Server TPS & MSPT
- Light level (block)
- Biome
- `z + c` for free camera
- `z + shift` for fake sneaking
- `l` for overlay light display
- `h + c` for config
